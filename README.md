# Bricksbuilder Markdown Block Plugin

Markdown Block Plugin for Bricks with Parsedown by erusev https://github.com/erusev/parsedown

## Manual

1. Download zip
2. Go to Plugins -> Install -> Install from file -> Select zip file
3. Enjoy :)
